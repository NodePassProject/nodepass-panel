# **App Name**: NodePass Manager

## Core Features:

- Instance Overview: Display a list of instances with key details like ID, type, status, URL, and traffic stats (TCP/UDP Rx/Tx).
- Instance Details: Show detailed information for a specific instance, including all available metrics and configuration.
- Create Instance: Form to create new instances, enforcing URL format validation and displaying feedback.
- Instance Control: Implement actions (start, stop, restart) on individual instances with visual confirmation of the status change.
- Delete Instance: Enable instance deletion with a confirmation step.
- Real-Time Events: Real-time updates of instance events (creation, updates, deletion, logs) via Server-Sent Events.
- AI-Powered Optimization: Use a generative AI tool to suggest optimal configurations (e.g., URL parameters) based on instance type and desired performance.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey stability and intelligence.
- Background color: Very light gray (#F5F5F5) for light mode and dark gray (#333333) for dark mode.
- Accent color: Vibrant cyan (#00BCD4) to draw attention to interactive elements.
- Use a clean, sans-serif font for optimal readability in both light and dark modes.
- Employ simple, modern icons to represent instance status and actions.
- Design a responsive layout that adapts seamlessly to different screen sizes.
- Incorporate subtle animations for transitions and feedback, enhancing user experience without being distracting.
